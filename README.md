# Deploying-to-AWS-using-CloudFormation

This project deploys a complete AWS environment using AWS CDK (Python).  
It includes a VPC, public subnets, an EC2 instance, and a MySQL RDS database.

## Project Structure
app.py  
InfraStack/networking_stack.py  
InfraStack/ec2_stack.py  
InfraStack/rds_stack.py  
requirements.txt  

## Prerequisites
- Python 3.10+  
- AWS CLI configured  
- AWS CDK installed globally (npm install -g aws-cdk)

## Setup
Create and activate virtual environment:
python -m venv .venv  
.venv\Scripts\activate  

Install dependencies:
pip install -r requirements.txt  

## Synth CloudFormation Templates
cdk synth NetworkingStack > networking.yml  
cdk synth Ec2Stack > ec2.yml  
cdk synth RdsStack > rds.yml  

## Deploy Stacks
Deploy all:  
cdk deploy --all  

Deploy individually:  
cdk deploy NetworkingStack  
cdk deploy Ec2Stack  
cdk deploy RdsStack  

## Destroy Stacks
cdk destroy --all  

## Summary
This CDK project automates creation of networking resources, an EC2 instance,  
and an RDS MySQL database using CloudFormation templates generated by AWS CDK.
